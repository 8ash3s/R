# Third Space Lounge Website

A warm, scrolling story page for a coffee/weed/beer third-space concept with rentable rooms, a jazz lounge, and an investor placement area.

## Quick start (non-coders)
1. Open `site/index.html` directly in your browser for a simple preview.
2. For the smoothest experience, run a tiny local server and visit http://localhost:8000:
   ```bash
   python -m http.server --directory site 8000
   ```
   Press `Ctrl+C` to stop the server when you are done.
